## 📂 Featured Projects

### 🔹 Lentera Mesh
LoRa mesh communication system using ESP32 for AIS-like telemetry, gateway communication, MQTT, OTA, and web configuration.

### 🔹 Welder HAR
Human Activity Recognition system for welding activity monitoring using React Native, ONNX, and Python-based federated / local workflow.

### 🔹 Self-Balancing Robot
ESP32-based balancing robot using IMU sensors, motor control, and PID tuning.

### 🔹 Aquaponics Controller
ESP32 + Blynk based aquaponics automation with pH sensor, feeder servo, OTA, and WiFiManager.

### 🔹 Gas Detection System
ESP32 project for gas monitoring using multiple sensors, LCD I2C, buzzer, and servo actuator.

### 🔹 Oil Drainer ESP32
PID-based motor speed control system with encoder feedback for automatic oil spinner / draining system.
